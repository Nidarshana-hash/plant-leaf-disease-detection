# 🌿 Plant Disease Detection System

## 📌 Overview

This project uses **Deep Learning (CNN)** to detect plant diseases from leaf images.
It helps farmers and researchers identify diseases quickly and accurately.

---

## 🚀 Features

* Image-based disease prediction
* Trained CNN model
* Easy-to-use interface
* High accuracy on test data

---

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* NumPy, Pandas
* Matplotlib
* (Optional) Flask / Streamlit

---

## 📂 Project Structure

```
plant-disease-detection/
│── data/            # dataset (not included)
│── models/          # trained model (not included)
│── notebooks/       # training & testing notebooks
│── src/             # main application code
│── static/          # images
```
